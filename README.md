# AIDO Test Fullstack Develope - Reza Agung Prasetya

## Overview

A technical test project for AIDO.

## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Project

```bash
local: npm run dev
build: npm run build
start: npm run start
```
